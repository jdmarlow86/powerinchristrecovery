# Power in Christ Recovery

A single-page website that shares the mission, rhythm, and recovery steps of the Power in Christ Recovery ministry.

## Getting Started

Open `index.html` in your preferred browser to explore the page locally.

```bash
open index.html # macOS
xdg-open index.html # Linux
start index.html  # Windows
```

## Project Structure

- `index.html` – The main page content.
- `styles.css` – Styling for the site, including typography, layout, and colors.
